There are three classes for each task. Each class has its own main method so they can use seperatly. 

Task 1:
Write a Java program, to take an HDFS Path as input and display all the files and sub-directories in that HDFS path.
Class: ListFile.TaskOne
Command line example: 

hadoop jar Session3Assignment1.jar ListFile.TaskOne /user
